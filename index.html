Difference between Window and Document Objects

Window Object:

  • 1. Window object is at the top level of the JavaScript object hierarchy. It is a global object in JavaScript and it is the root object of the DOM (Document Object Model).
  • 2. It represents the browser window in which you are viewing the webpage.
  • 3. Properties related to the window are stored in this object.
  • 4. It is loaded before the document because it contains the document.
  • 5. You can access document object properties inside the window.

Document Object:

  • 1. The document object lies inside the window object, and we use it for manipulation within the document.
  • 2. It represents the document loaded inside the window or browser.
  • 3. Properties related to the document are stored in the document object.
  • 4. It is loaded after the window because the window contains the document.
  • 5. You cannot access window object properties in a document.
